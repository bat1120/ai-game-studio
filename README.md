# 리턴 레코드: 잊힌 항해자들 (Return Record)

> AI 에이전트들이 시장분석·세계관·캐릭터·기획·UI·개발·QA·배포까지 자율 제작한 2D 액션 게임.

## 무엇인가

폐허가 된 관제함 **아르카디아 호**에서, 기억을 잃은 채 매번 돌아오는 항해자 **번호-0**이 되어 폐역을 돌파하는 짧고 밀도 높은 스토리 액션 게임입니다. 죽어도 열차에 남은 기록 조각이 돌아오기에, 잃을수록 다시 강해집니다.

장르는 2D 캐릭터 액션 로그라이크 + 비주얼노벨(VN) 하이브리드이며, 단일 HTML5 canvas 파일로 구현되어 있습니다.

## 어떻게 만들어졌나

이 게임은 **AI Game Studio** 파이프라인의 산출물입니다. 사람이 일일이 코딩한 것이 아니라, 여러 AI 에이전트가 회의 형태로 역할을 나누어 — 시장분석, 세계관 설정, 캐릭터 설계, 게임 기획, UI, 개발, QA, 배포까지 — 전 과정을 자율적으로 진행해 만든 결과물입니다.

`store.html`에는 이 제작 과정에서 나온 품질 평가 점수가 그대로 기록되어 있습니다(비전 83 / 재미 64 / 스토리 72, 기능 통과, 출시 적합 아니오). 마케팅된 완성품이 아니라 AI 자율 제작 실험의 스냅샷으로 보면 됩니다.

## 게임 내용

- **진행 구조**: 타이틀 → 인트로 컷신(VN) → 스테이지 전투 → 보스전 → 스토리(VN) → 3택 업그레이드 → 다음 구간 반복.
- **스테이지**: 폐역 / 선로 / 객차 — 각 2웨이브, 적은 소수 정예(차장 conductor 근접 / 감시자 watcher 원거리 / 관문 gate).
- **보스 3종**: 적루왕 브락(돌진·강타), 감찰관 아델(공백 폭발·스킬 봉인), 무명 승객(플레이어 스킬 복제·이름 도둑질). 각 보스는 HP 50% 이하에서 페이즈 2로 전환됩니다.
- **업그레이드**: 스테이지 클리어마다 풀에서 3개를 뽑아 1개 선택(칼날 사거리, 적중 시 기억 게이지, 방벽 강화, 리턴 복제 시 무적, 최대 체력 +30, 전 스킬 쿨다운 -15%, 기억 게이지 최대 +30).
- **세이브**: 닉네임/비밀번호 계정 또는 게스트로 시작하며, 최고 점수·클리어 여부·복구한 기억 수가 브라우저 `localStorage`에 저장됩니다.

### 조작법

| 키 | 동작 |
|---|---|
| `A` / `D` (또는 ←/→) | 이동 |
| `J` | 기본 공격 (3타 콤보, 3타째 적중 시 기억 게이지 획득) |
| `K` | 잔향 칼날 (전방 연속 베기, 쿨다운 6초) |
| `L` | 되감기 방벽 (방어막, 피격 시 충격파, 쿨다운 10초) |
| `U` | 리턴 레코드 (직전 행동을 복제·강화, 쿨다운 8초) |
| `Shift` | 회피/대시 (무적 프레임, 완벽 회피 시 스킬 쿨다운 감소) |
| `Q` | 동료 지원 (세이프: 복구 구역 / 루나: 잔향 해독) |
| `Space` / `Enter` | 출격·대화 넘김·확인 |

점프는 없습니다. 마우스 클릭으로 타이틀 진행·대화 넘김·업그레이드 선택이 가능합니다.

> **리턴 레코드(U)** 가 시그니처 기술입니다. 직전 행동(공격/회피/스킬K/스킬L)이 무엇이었는지에 따라 분신 강타, 순간이동 반격, 칼날 재시전, 방벽 재생성으로 다르게 발동합니다.

## 기술 스택

- 순수 **HTML5 + Canvas 2D + 바닐라 JavaScript**. 빌드 도구·외부 라이브러리·프레임워크 없음.
- 게임 전체가 `index.html` 한 파일 안에 인라인 스크립트로 구현(델타타임 기반 게임 루프, 파티클·스크린셰이크·히트스톱 등 자체 연출 포함).
- 저장은 브라우저 `localStorage`.
- 캐릭터/적/배경 스프라이트는 `assets/`의 PNG. 이미지 로드 실패 시 도형으로 폴백 렌더링하도록 안전 처리되어 있습니다.
- 정적 호스팅은 **GitHub Pages**.

## 핵심 파일 구조

```
ai-game-studio/
├─ index.html              # 게임 본체 (canvas + 전체 게임 로직·VN·전투·보스·세이브)
├─ store.html              # 소개/스토어 페이지 (세계관·캐릭터·특징·품질 평가)
├─ shot0.png ~ shot2.png   # 스토어용 스크린샷
└─ assets/
   ├─ char0.png ~ char5.png       # 캐릭터 초상 (번호-0, 세이프, 루나, 아델, 브락, 무명 승객)
   ├─ actor0/3/4/5.png            # 인게임 액터 스프라이트 (플레이어·보스)
   ├─ enemy_conductor.png         # 일반 적: 차장
   ├─ enemy_watcher.png           # 일반 적: 감시자
   ├─ environment_gate.png        # 일반 적: 관문
   └─ background.png              # 전투 배경
```
